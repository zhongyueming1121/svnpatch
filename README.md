# svnpatch
#svn和git下快速增量包打包工具\n
#功能
[1] checkout指定版本代码  
[2] 根据提交记录抽取文件打包成增量包  
[3] 支持maven和ant打包  
[4] svn支持自动读取本机账号密码  
[5] 支持配置文件记忆  

#待支持功能
[1] git  
[2] 根据patch文件打增量包
